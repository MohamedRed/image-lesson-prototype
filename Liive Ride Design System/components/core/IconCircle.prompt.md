First-line one-sentence: A tinted circular icon badge — the recurring feature glyph for cards, list rows and callouts.

```jsx
<IconCircle color="accent"><i data-lucide="car" /></IconCircle>
<IconCircle color="danger" filled size={32}><i data-lucide="shield" /></IconCircle>
```

`color` maps to the semantic palette; `filled` gives a solid white-glyph badge; `size` sets the diameter (default 44).
